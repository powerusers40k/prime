# prime
alternative to the web

# supports

- any DOMAINS (for free)
- tweakui.css: update the interface
- unlimited amount of subdomains
- support for making HTML, CSS, Lua (no JS)
- supports Go backend as microservices

# available sites

- **crawler.tri:** crawls websites, respects restricts.txt and saves entire code
- **play.vids:** upload and play videos, no Ads.
